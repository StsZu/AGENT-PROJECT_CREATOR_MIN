# ACCESS (template)

**ЦЕ ШАБЛОН. Реальний файл називається `ACCESS.md` і знаходиться у `.gitignore`.**

Перевірка, що реальний `ACCESS.md` ніколи не потрапляв у git:

```bash
git log --all -- ACCESS.md
# має повернути порожньо
```

Якщо повернуло щось — секрети вже у git history, треба ротувати все, що там було, і чистити історію.

## Як цим користуватись

1. Скопіювати цей файл у `ACCESS.md` (без `.example`).
2. Заповнити реальними значеннями.
3. Переконатись, що `ACCESS.md` уже в `.gitignore`.
4. Ніколи не комітити `ACCESS.md`.
5. При оновленні пароля — оновити дату і повідомити власника.
